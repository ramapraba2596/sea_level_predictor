🌊 Sea Level Predictor

This project focuses on analyzing historical data of global sea level changes to understand long-term trends and predict future patterns.

The dataset contains records of sea level measurements collected over many years. The main goal of this project is to study how sea levels have changed over time and estimate how they are likely to rise in the future.

The project begins with visualizing the data using a scatter plot to observe overall trends. It then applies linear regression to calculate the line of best fit for the entire dataset, helping to understand the general increase in sea level.

Additionally, a separate analysis is performed using more recent data (from the year 2000 onwards) to capture current trends more accurately. Both trends are extended to predict sea level changes up to the year 2050, providing insights into future scenarios.

These results help in understanding the impact of environmental changes and the rate at which sea levels are rising.

🔧 Technologies Used

- Python
- Pandas
- Matplotlib
- SciPy

📁 Dataset

- Global Sea Level Change Dataset

🎯 Key Features

- Scatter plot visualization of sea level data
- Trend analysis using linear regression
- Future prediction up to the year 2050
- Comparison of historical and recent trends
- Insight into long-term environmental changes

This project demonstrates how data analysis techniques can be used to study real-world problems and make meaningful predictions.
